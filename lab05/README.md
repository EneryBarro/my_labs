# Загальне завдання

1. Визначити, чи є ціле 6-значне число «щасливим» квитком («щасливий квиток» - квиток, в якому сума першої половини чисел номера дорівнює сумі другої половини). Наприклад, білет з номером 102300 є щасливим, бо 1 + 0 + 2 = 3 + 0 + 0.
